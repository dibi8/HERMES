### Vòng lặp tinh chỉnh prompt

Trọng tâm của hệ thống là cơ chế tinh chỉnh prompt. Người dùng có thể nhập một ý tưởng mơ hồ, chẳng hạn như "một thành phố cyberpunk". Mô hình GPT-4o sẽ mở rộng ý tưởng này thành một prompt chi tiết và có cấu trúc. Quá trình này bao gồm việc thêm các điều kiện ánh sáng, góc máy, bảng màu và tài liệu tham khảo về phong cách. Kho lưu trữ cung cấp các đoạn mã minh họa cách tự động hóa quá trình mở rộng này theo chương trình.

```python
import openai

client = openai.OpenAI()

def expand_prompt(original_idea):
    system_prompt = """
    Bạn là một kỹ sư prompt chuyên nghiệp cho các mô hình tạo hình ảnh. 
    Nhiệm vụ của bạn là lấy một khái niệm đơn giản và mở rộng nó thành một mô tả 
    chi tiết phù hợp cho việc tạo hình ảnh bằng GPT-4o. Bao gồm các chi tiết về:
    - Ánh sáng (ví dụ: khối, neon, giờ vàng)
    - Góc máy (ví dụ: góc thấp, quay từ trên cao, cận cảnh)
    - Phong cách (ví dụ: chân thực, tranh sơn dầu, render 3D)
    - Các vật thể và kết cấu cụ thể
    """
    
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "system", "content": system_prompt},
            {"role": "user", "content": f"Mở rộng ý tưởng này: {original_idea}"}
        ],
        temperature=0.7
    )
    return response.choices[0].message.content

# Ví dụ sử dụng
prompt = "expand_prompt('một khu vườn tương lai')"
print(prompt)
```

### Phân tích trực quan và phản hồi

Một chức năng quan trọng khác là phân tích các hình ảnh đã tạo. GPT-4o có thể nhìn vào một hình ảnh và phê bình nó dựa trên prompt ban đầu. Điều này tạo ra một hệ thống vòng kín nơi mô hình xác định các điểm không khớp—chẳng hạn như thiếu các yếu tố hoặc màu sắc sai—và đề xuất các điều chỉnh đối với prompt. Kho lưu trữ bao gồm các kịch bản tạo điều kiện cho vòng lặp phản hồi này, cho phép cải tiến lặp đi lặp lại mà không cần sự can thiệp thủ công.

```python
def analyze_image(image_url, original_prompt):
    system_prompt = """
    Phân tích hình ảnh được cung cấp so với prompt ban đầu. 
    Xác định bất kỳ phần tử nào bị thiếu, sai lệch phong cách hoặc lỗi.
    Cung cấp điểm số từ 1-10 và đề xuất các chỉnh sửa cụ thể cho prompt.
    """
    
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "system", "content": system_prompt},
            {"role": "user", "content": [
                {"type": "text", "text": f"Prompt: {original_prompt}"},
                {"type": "image_url", "image_url": {"url": image_url}}
            ]}
        ]
    )
    return response.choices[0].message.content
```

### Đánh giá chất lượng hình ảnh

Sử dụng điểm số CLIP (Huấn luyện trước tương phản ngôn ngữ-hình ảnh), chúng tôi đo lường độ tương đồng ngữ nghĩa giữa prompt và hình ảnh.

```python
from transformers import CLIPProcessor, CLIPModel
import torch

model = CLIPModel.from_pretrained("openai/clip-vit-base-patch32")
processor = CLIPProcessor.from_pretrained("openai/clip-vit-base-patch32")

def calculate_clip_score(text, image_path):
    inputs = processor(text=[text], images=[image_path], return_tensors="pt", padding=True)
    outputs = model(**inputs)
    logits_per_image = outputs.logits_per_image
    return logits_per_image.item()

# Ví dụ: So sánh điểm số
score_raw = calculate_clip_score("mèo", "raw_cat.jpg")
score_expanded = calculate_clip_score("mèo vằn cam lông xù ngồi trên bệ cửa sổ với mưa bên ngoài", "expanded_cat.jpg")
print(f"Điểm thô: {score_raw}, Điểm mở rộng: {score_expanded}")
```

## Sử dụng nâng cao: Triển khai sản xuất

Triển khai Awesome Gpt4O Images trong môi trường sản xuất đòi hỏi xử lý lỗi mạnh mẽ, giới hạn tốc độ (rate limiting) và các chiến lược lưu trữ đệm (caching).

### Triển khai giới hạn tốc độ

Các API của OpenAI áp dụng giới hạn tốc độ. Để ngăn chặn lỗi 429, hãy triển khai cơ chế quay lại theo cấp số nhân (exponential backoff).

```python
import time
import requests

def api_request_with_retry(url, headers, payload, max_retries=5):
    for attempt in range(max_retries):
        try:
            response = requests.post(url, json=payload, headers=headers)
            response.raise_for_status()
            return response.json()
        except requests.exceptions.HTTPError as e:
            if e.response.status_code == 429:
                wait_time = 2 ** attempt
                print(f"Bị giới hạn tốc độ. Đang chờ {wait_time}s...")
                time.sleep(wait_time)
            else:
                raise e
    raise Exception("Vượt quá số lần thử lại tối đa")
```

### Lưu trữ đệm cho các prompt đã tạo

```python

CACHE_FILE = "prompts_cache.json"

def load_cache():
    if os.path.exists(CACHE_FILE):
        with open(CACHE_FILE, 'r') as f:
            return json.load(f)
    return {}

def save_cache(cache):
    with open(CACHE_FILE, 'w') as f:
        json.dump(cache, f)

cache = load_cache()
if original_idea not in cache:
    cache[original_idea] = expand_prompt(original_idea)
    save_cache(cache)

refined_prompt = cache[original_idea]
```
