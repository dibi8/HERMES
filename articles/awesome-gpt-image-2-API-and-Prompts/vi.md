### Lớp Kỹ Thuật Lời Nhắc (Prompt Engineering)

Bước đầu tiên trong quy trình làm việc là xây dựng lời nhắc. Kho lưu trữ cung cấp một thư viện các lời nhắc đã được tối ưu hóa, đóng vai trò là mẫu. Người dùng có thể chọn một lời nhắc cơ sở và sửa đổi các tham số như phong cách, ánh sáng, bố cục và chi tiết chủ thể. Ví dụ, người dùng có thể bắt đầu với một lời nhắc chung chung cho một "thành phố tương lai" và tinh chỉnh nó bằng cách sử dụng các mẫu được cung cấp để chỉ định "phong cách cyberpunk," "ánh sáng neon" và "khí hậu mưa."

```python
# Ví dụ về việc xây dựng lời nhắc sử dụng cấu trúc của thư viện
prompt_template = {
    "subject": "a futuristic city",
    "style": "cyberpunk",
    "lighting": "neon",
    "atmosphere": "rainy",
    "quality_tags": ["8k resolution", "highly detailed", "photorealistic"]
}

final_prompt = f"{prompt_template['subject']}, {prompt_template['style']}, {prompt_template['lighting']}, {prompt_template['atmosphere']}, {', '.join(prompt_template['quality_tags'])}"
print(final_prompt)
```

### Xử Lý Phía Máy Chủ Và Xử Lý Phản Hồi

Cổng API xử lý yêu cầu bằng cách chuyển tiếp nó đến mô hình đã chọn. Mô hình chạy quá trình khuếch tán, liên tục khử nhiễu từ nhiễu ngẫu nhiên để tạo thành một hình ảnh mạch lạc dựa trên hướng dẫn văn bản. Sau khi quá trình tạo hoàn tất, API trả về một phản hồi chứaeither URL của hình ảnh đã tạo hoặc chính dữ liệu hình ảnh ở dạng mã hóa base64.

```javascript
// Lấy hình ảnh từ API trong Node.js
const axios = require('axios');

async function generateImage(promptData) {
  try {
    const response = await axios.post('https://api.example.com/generate', promptData, {
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
      },
      responseType: 'arraybuffer' // Để xử lý dữ liệu hình ảnh nhị phân
    });

    // Lưu hình ảnh
    const fs = require('fs');
    fs.writeFileSync('output.png', Buffer.from(response.data));
    console.log('Image saved successfully.');
  } catch (error) {
    console.error('Error generating image:', error.message);
  }
}

generateImage({
  prompt: "a futuristic city, cyberpunk, neon, rainy",
  width: 1024,
  height: 1024
});
```

### Tái Tạo Và Quản Lý Hạt Giống (Seed)

Một khía cạnh quan trọng của việc tạo hình ảnh chuyên nghiệp là khả năng tái tạo. Kho lưu trữ nhấn mạnh việc sử dụng các giá trị hạt giống (seed). Bằng cách đặt một hạt giống cụ thể, người dùng có thể tạo ra các biến thể của cùng một hình ảnh với những thay đổi nhỏ về tham số, đảm bảo tính nhất quán trong các dự án thiết kế. Hướng dẫn cung cấp các kịch bản để quản lý hạt giống trên nhiều lô tạo hình ảnh.

```bash
# Sử dụng cURL để tạo nhiều biến thể với hạt giống cố định
for i in {1..5}; do
  curl -X POST "https://api.example.com/generate" \
  -H "Content-Type: application/json" \
  -d "{
    \"prompt\": \"a futuristic city\",
    \"seed\": 42,
    \"variation_strength\": $i
  }" > output_var_${i}.png
done
```

Cách tiếp cận mô-đun này cho phép các nhà phát triển tích hợp tạo hình ảnh một cách liền mạch vào các quy trình lớn hơn, chẳng hạn như hệ thống tạo nội dung tự động, công cụ thiết kế hoặc các ứng dụng web tương tác.

### Tích Hợp Plugin WordPress

Đối với người sáng tạo nội dung sử dụng WordPress, kho lưu trữ cung cấp hướng dẫn để tạo các plugin tùy chỉnh cho phép tạo hình ảnh trực tiếp từ trình soạn thảo.

```php
// Ví dụ shortcode cơ bản của WordPress
function generate_ai_image($atts) {
    $atts = shortcode_atts(array(
        'prompt' => '',
        'size' => '1024x1024'
    ), $atts);

    $api_url = "https://api.provider.com/v1/images/generations";
    $data = array(
        "prompt" => $atts['prompt'],
        "size" => $atts['size']
    );

    $response = wp_remote_post($api_url, array(
        'body' => json_encode($data),
        'headers' => array(
            'Content-Type' => 'application/json',
            'Authorization' => 'Bearer ' . get_option('ai_api_key')
        )
    ));

    if (is_wp_error($response)) {
        return "Error generating image.";
    }

    $body = json_decode(wp_remote_retrieve_body($response), true);
    return '<img src="' . $body['data'][0]['url'] . '" alt="AI Generated Image">';
}
add_shortcode('ai_image', 'generate_ai_image');
```

Các nhà thiết kế có thể tích hợp API vào Figma bằng cách sử dụng các plugin tùy chỉnh, cho phép tạo tài sản theo thời gian thực bên trong bảng vẽ thiết kế.

// Ví dụ API Plugin Figma
figma.ui.onmessage = async (msg) => {
  if (msg.type === 'generateImage') {
    const response = await fetch('https://api.provider.com/v1/images/generations', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer ' + figma.env.get('API_KEY')

      body: JSON.stringify({ prompt: msg.prompt })
    });
    
    const data = await response.json();
    figma.ui.postMessage({ type: 'imageGenerated', url: data.data[0].url });
  }
};
```

### Thực Hành Bảo Mật Tốt Nhất

*   **Giới Hạn Tốc Độ (Rate Limiting):** Triển khai giới hạn tốc độ để ngăn chặn lạm dụng.
*   **Xác Thực Đầu Vào:** Làm sạch tất cả các đầu vào của người dùng để ngăn chặn các cuộc tấn công chèn mã.

# Ví dụ cấu hình giới hạn tốc độ trong Nginx
limit_req_zone $binary_remote_addr zone=mylimit:10m rate=10r/s;

location /api/ {
    limit_req zone=mylimit burst=20 nodelay;
    proxy_pass http://backend;
}
```
