Để minh họa cách các prompt hoạt động trong thực tế, hãy xem tập lệnh Python sau đây minh họa việc tải một prompt từ thư viện và gửi nó đến LLM:

```python
import requests
import json

# Xác định điểm cuối API cho LLM
API_URL = "https://api.your-llm-provider.com/v1/completions"

# Tải một prompt từ thư viện Awesome Nano Banana Pro
def load_prompt(prompt_id):
    # Trong tình huống thực tế, điều này sẽ truy xuất từ kho cục bộ hoặc API
    prompts = {
        "summarizer": "Tóm tắt đoạn văn bản sau trong không quá 100 từ:",
        "coder": "Viết hàm Python để tính giai thừa của một số:"
    }
    return prompts.get(prompt_id, "ID prompt không xác định")

# Ví dụ sử dụng
user_input = "Con cáo nâu nhanh nhẹn nhảy qua chú chó lười biếng."
prompt_template = load_prompt("summarizer")
full_prompt = f"{prompt_template} {user_input}"

# Gửi prompt đến LLM
headers = {"Content-Type": "application/json"}
data = {
    "model": "nano-banana-pro-v1",
    "prompt": full_prompt,
    "max_tokens": 150
}

response = requests.post(API_URL, headers=headers, json=data)
result = response.json()

print(result['choices'][0]['text'])
```

Ví dụ này cho thấy việc tích hợp các prompt vào ứng dụng hiện có đơn giản như thế nào. Hàm `load_prompt` truy xuất một mẫu được xác định trước, sau đó được kết hợp với đầu vào của người dùng trước khi gửi đến LLM. ## Tích hợp với các công cụ phổ biến

Một tích hợp phổ biến là với LangChain, một khung làm việc để phát triển các ứng dụng được cung cấp bởi LLMs. Các trừu tượng chuỗi và tác nhân của LangChain giúp dễ dàng đưa các prompt từ thư viện Awesome Nano Banana Pro vào các quy trình làm việc phức tạp. Dưới đây là ví dụ về cách sử dụng một prompt từ thư viện bên trong một chuỗi LangChain:

```python
from langchain.prompts import PromptTemplate
from langchain.llms import OpenAI
from awesome_nano_banana_prompts import load_prompt_template

# Khởi tạo LLM
llm = OpenAI(model_name="gpt-3.5-turbo")

# Tải một mẫu prompt từ thư viện
template = load_prompt_template("creative_writer")
prompt = PromptTemplate.from_template(template)

# Tạo một chuỗi
chain = prompt | llm

# Thực thi chuỗi
result = chain.invoke({"topic": "space exploration"})
print(result)
```

Một tích hợp phổ biến khác là với thư viện Transformers của Hugging Face, được sử dụng rộng rãi cho các nhiệm vụ xử lý ngôn ngữ tự nhiên. Bằng cách chuyển đổi các prompt thành các tập dữ liệu hoặc tokenizer của Hugging Face, người dùng có thể tận dụng sức mạnh của các mô hình đã huấn luyện sẵn cùng với các prompt được tuyển chọn.

```python
from transformers import pipeline

# Tải một pipeline tạo văn bản
generator = pipeline('text-generation', model='nano-banana-pro-model')

# Xác định một prompt từ thư viện
prompt_text = """
Viết một câu chuyện ngắn về một robot học cách vẽ tranh.
Bao gồm các chi tiết giác quan và chiều sâu cảm xúc.
"""

# Tạo văn bản
output = generator(prompt_text, max_length=200, num_return_sequences=1)
print(output[0]['generated_text'])
```

Các tích hợp này nhấn mạnh tính linh hoạt của thư viện Awesome Nano Banana Pro Prompts. ## Sử dụng nâng cao: Triển khai sản xuất

Đối với các doanh nghiệp và ứng dụng quy mô lớn, việc triển khai Awesome Nano Banana Pro Prompts đòi hỏi sự cân nhắc cẩn thận về khả năng mở rộng, bảo mật và giám sát. Thư viện hỗ trợ containerization và các công cụ điều phối như Kubernetes, khiến nó phù hợp cho các môi trường sản xuất phức tạp.

Một thực tiễn tốt là triển khai lớp bộ nhớ đệm cho các prompt được sử dụng thường xuyên. Vì nhiều prompt là tĩnh, việc lưu trữ kết quả của các cuộc gọi LLM trong bộ nhớ đệm có thể giảm đáng kể chi phí và cải thiện thời gian phản hồi. Dưới đây là ví dụ về cách triển khai bộ nhớ đệm bằng Redis trong một ứng dụng Python:

```python
import redis
import hashlib
from langchain.llms import OpenAI

# Kết nối đến Redis
redis_client = redis.Redis(host='localhost', port=6379, db=0)

# Hàm tạo khóa bộ nhớ đệm
def get_cache_key(prompt, model):
    raw_key = f"{prompt}:{model}"
    return hashlib.sha256(raw_key.encode()).hexdigest()

# Wrapper cho cuộc gọi LLM với bộ nhớ đệm
def get_llm_response_with_cache(prompt, model="gpt-3.5-turbo"):
    cache_key = get_cache_key(prompt, model)
    cached_response = redis_client.get(cache_key)
    
    if cached_response:
        return cached_response.decode('utf-8')
    
    # Gọi LLM
    llm = OpenAI(model_name=model)
    response = llm(prompt)
    
    # Lưu vào bộ nhớ đệm với TTL 24 giờ
    redis_client.setex(cache_key, 86400, response)
    
    return response
```

Bảo mật là một khía cạnh quan trọng khác của việc triển khai sản xuất. Đảm bảo rằng các khóa API và dữ liệu nhạy cảm được bảo vệ là ưu tiên hàng đầu.
